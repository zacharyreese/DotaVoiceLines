How To Bind Voice Lines

Quick guide on how to bind voice lines to keystrokes using the autoexec.cfg file and console commands

Step-by-Step Guide: Creating an Autoexec File

Step 1: Locate Your Dota 2 Directory

First, you need to find where Dota 2 is installed on your computer. The default locations are:

  • Windows: C:\Program Files (x86)\Steam\steamapps\common\dota 2 beta\game\dota\cfg
  • Mac: ~/Library/Application Support/Steam/steamapps/common/dota 2 beta/game/dota/cfg
  • Linux: ~/.local/share/Steam/steamapps/common/dota 2 beta/game/dota/cfg

Note: If you installed Steam in a custom location, navigate to that location instead.

Step 2: Navigate to the CFG Folder

Once you've found your Dota 2 installation, navigate to:

dota 2 beta/game/dota/cfg

This is where all configuration files are stored.

Step 3: Create the Autoexec File

In the cfg folder, create a new text file and name it:

autoexec.cfg

Important: Make sure the file extension is .cfg and not .txt. You may need to enable "Show file extensions" in your file explorer settings to see and change this.

üí° Tip: On Windows, you can create this file by right-clicking in the folder, selecting "New" ‚Üí "Text Document", then renaming it to autoexec.cfg (including the extension).

Step 4: Add Your Voice Line Binds

Open the autoexec.cfg file with any text editor (Notepad, Notepad++, TextEdit, etc.) and add your binds. Here's the format:

bind "KEY" "chatwheel_say VOICE_LINE_ID"

Example binds:

// Basic voice line binds
bind "KP_0" "chatwheel_say 401003"    // Bind Ephey: 'Sorry!' to Num Pad 0
bind "M" "chatwheel_say 401275"       // Bind Slacks: 'Owwww!' to keyboard letter M
bind "UPARROW" "chatwheel_say 401683" // Bind Tsunami: 'Stop!' to Up Arrow
bind "F4" "chatwheel_say 401258"      // Bind Fogged: 'Not bad at all' to F4

// You can also add comments using //
bind "F2" "chatwheel_say 123"  // This is a comment

üí° Tip: Browse the voice lines on the home page to find the IDs you want to use!

üü• Note: You can only bind voice lines that you own.

Step 5: Save the File

Save the autoexec.cfg file and close your text editor.

Step 6: Enable the Developer Console (If Not Already Enabled)

In Dota 2:

  1. Go to Settings ‚Üí Advanced Options
  2. Check the box for "Enable Developer Console"
  3. Set a key to open the console (default is the ` key)

Step 7: Test Your Binds

Launch Dota 2 and open the console (press ` or your chosen console key). Type:

exec autoexec.cfg

This will manually execute your autoexec file. Your binds should now work!

Note: The autoexec file should load automatically each time you start Dota 2, but you can always run this command to reload it after making changes.

Common Key Names for Binds

  • Numpad: KP_1, KP_2, KP_3, etc.
  • Function Keys: F1, F2, F3, etc.
  • Letters: A, B, C, etc.
  • Special Keys: SPACE, SHIFT, CTRL, ALT, ENTER, TAB, ESCAPE
  • Mouse Buttons: MOUSE3, MOUSE4, MOUSE5

Troubleshooting

  • Binds not working? Make sure the file is named exactly autoexec.cfg and is in the correct folder.
  • File extension wrong? Enable "Show file extensions" in Windows Explorer or Finder to ensure the file isn't actually autoexec.cfg.txt.
  • Need to reload? Open console and type exec autoexec.cfg to reload your configuration without restarting Dota 2.
  • Key already in use? Check your in-game keybinds to make sure you're not creating conflicts.
